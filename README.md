# Ad Click Fraud Detection

Based on: Alzahrani, Aljabri & Mohammad (IEEE Access, 2025)

## Dataset
Download advertising.csv from:
https://www.kaggle.com/datasets/gabrielsantello/advertisement-click-on-ad

Place it in the project root before running.

## Requirements
pip install pandas numpy scikit-learn matplotlib seaborn

## Run
python3 adclick.py
